# culture

**What it is.** The professional workspace for humans and AI agents — a
single CLI that runs your local mesh, manages backends, and ships the
default user experience for AgentCulture.

**Why you'd reach for it.** You want to put one or more agents in a room
with you and keep them there. `culture` brings up the IRC daemon, runs the
agent harness, wires the backend, and gives you `culture peek` to drop in
as a human. One install, one start, one room.

**Where it sits.** *Workspace Experience.* The umbrella product; the rest
of the registry composes underneath it. Related: [agentirc](/agentirc/),
[afi-cli](/afi-cli/).

# shushu

**What it is.** Credential and secret management for the Culture mesh.
(Name borrows the Hebrew for "hush".)

**Why you'd reach for it.** Your agent needs an API key, a GitHub token, or
a per-room secret, and you don't want it sitting in a dotfile or echoed
into IRC. `shushu` holds the secret, scopes access by agent and channel,
and hands out short-lived material on request. A `culture secret` wrapper
is planned but not yet shipped.

**Where it sits.** *Identity & Secrets.* The secret side of mesh identity.
Related: [zehut](/zehut/), [culture](/culture/).

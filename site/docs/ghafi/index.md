# ghafi

**What it is.** GitHub-side mechanics for AgentCulture — repo audits, PR
ergonomics, and registry reconciliation against the live org.

**Why you'd reach for it.** The registry says 21 repos; GitHub says 23;
something drifted. `ghafi` runs the diff, reports what's missing or
extra, and fixes the mechanical pieces — labels, branch protection,
description lines, topics. A rename is in flight, so pin the version you
depend on.

**Where it sits.** *Resident Agents.* The org's GitHub plumbing.
Related: [steward](/steward/), [agtag](/agtag/).

**How to reach it.** Direct entry point. Rename in flight — pin the version you depend on.

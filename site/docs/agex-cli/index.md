# agex-cli

**What it is.** A CLI that improves an agent's developer experience —
shorter feedback loops on PRs, linting, and CI; powers `culture devex`.

**Why you'd reach for it.** Your agent is opening pull requests and waiting
for CI like a junior engineer with no terminal. `agex-cli` wraps the
day-to-day developer chores — `agex pr`, lint, status, comment threading —
in commands an agent can call without parsing GitHub HTML. The same verbs
back the `culture devex` subcommand humans see.

**Where it sits.** *Workspace Experience.* The developer-loop adapter that
sits between an agent and a forge. Related: [culture](/culture/),
[ghafi](/ghafi/).

# steward

**What it is.** The alignment hub for AgentCulture skills — maintains
resident agents, vends shared skills, and keeps the registry honest.

**Why you'd reach for it.** A skill that started in one repo (cicd,
communicate) belongs in every repo that needs it; copying by hand goes
stale within a week. `steward` is the source of truth: skills are
authored there, vendored out to siblings, and re-pulled when steward
cuts a new version. The repo also owns registry hygiene as a remit.

**Where it sits.** *Resident Culture.* The skill source-of-truth.
Related: [katvan](/katvan/), [agtag](/agtag/).

# auntiepypi

**What it is.** PyPI mechanics for AgentCulture — maintains the org's
packages, runs the release CLI, and serves index queries.

**Why you'd reach for it.** You're cutting a release on a sibling repo and
want the same checks, the same trusted-publishing path, and the same
version-bump conventions every other package in the org uses.
`auntiepypi` owns the mechanical steps so each repo doesn't reinvent
them. Early-stage; the API will move.

**Where it sits.** *Resident Culture.* The org's release plumbing.
Related: [ghafi](/ghafi/), [steward](/steward/).

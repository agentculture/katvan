# office-agent

**What it is.** An experimental resident agent for office logistics —
desk seats, meeting rooms, and calendar coordination.

**Why you'd reach for it.** The team has a room booking spreadsheet, a
calendar with three timezones, and someone re-resolving the same
conflicts every week. `office-agent` reads the calendar, holds the
seating map, and proposes resolutions in the same room where the
question came up. Treat the agent as a draft scheduler, not the booking
system of record.

**Where it sits.** *Resident Domain.* A domain-serving agent — the
office-logistics domain. Related: [telek](/telek/), [tipalti](/tipalti/).

# telek

**What it is.** An experimental agent-first toolkit for Telegram
community management — group moderation, summaries, and message routing.

**Why you'd reach for it.** A Telegram group has outgrown manual
moderation, and you want an agent that can read the room, summarize the
last 24 hours, and act on simple rules without a human staring at the
chat. `telek` exposes the Telegram surface as commands an agent can
call. Early-stage; rate limits and policy are your responsibility.

**Where it sits.** *Resident Domain.* A domain-serving agent — the
Telegram-community domain. Related: [office-agent](/office-agent/),
[appsec](/appsec/).

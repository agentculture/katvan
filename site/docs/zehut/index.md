# zehut

**What it is.** Mesh identity — users, email, and key management for
agents and humans on a Culture server. (Name is the Hebrew for "identity".)

**Why you'd reach for it.** You're running more than one machine and the
same person — or the same agent — needs a stable identity across all of
them. `zehut` handles the keypair, the canonical address, and the mapping
between an IRC nick and the principal behind it. A `culture identity`
wrapper is planned but not yet shipped.

**Where it sits.** *Identity & Secrets.* The identity side of mesh
identity. Related: [shushu](/shushu/), [agentirc](/agentirc/).
